## 안녕하세요, 홍정모입니다.

MIDASIT에서 C++ 기반 CAE 응용소프트웨어를 개발하고 있습니다.

**특정 도메인의 문제를 AI/LLM으로 해결하는 시스템을 설계하고 구현하는 데 깊은 관심을 갖고 있습니다.**

엔지니어링 소프트웨어 개발 경험을 바탕으로, 도메인 전문 지식이 필요한 영역에서 AI가 실질적인 가치를 만들어내는 시스템을 구현하는 역량을 쌓아가고 있습니다.

---

## 관심 분야

도메인 특화 AI 시스템을 구현하기 위해 다음 영역들을 집중적으로 탐구하고 있습니다.

- **Domain-Driven RAG** — 전문 도메인의 용어와 맥락을 정확히 반영하는 검색 증강 생성 시스템
- **LLM Fine-tuning** — 도메인 특화 Tool Calling 등 특수 목적에 맞춘 모델 최적화
- **Multi-Agent Systems** — 복잡한 도메인 워크플로우를 자동화하는 에이전트 시스템 설계

---

## 주요 프로젝트

### [DesignCodeRAG](https://github.com/Gabriel-Hong/DesignCodeRAG)

한국 건축법규라는 전문 도메인에 특화된 RAG 시스템입니다. Elasticsearch 하이브리드 검색(BM25 + Dense Vector)을 활용하여 도메인 고유 용어와 맥락을 정확히 반영한 답변을 생성합니다.

### [qlora-function-calling](https://github.com/Gabriel-Hong/qlora-function-calling)

구조공학 API의 Tool Calling을 정확히 수행하도록 Qwen2.5 모델을 QLoRA로 파인튜닝한 프로젝트입니다. 도메인 특화 MCP Tool Calling이 가능한 경량 모델을 구현했습니다.

### [MultiAgentSystem4Material](https://github.com/Gabriel-Hong/MultiAgentSystem4Material)

Jira 이슈 분석부터 C++ 코드 수정, PR 생성까지 일련의 개발 워크플로우를 자동화하는 멀티에이전트 시스템입니다. 실무 도메인의 반복 작업을 에이전트로 해결한 사례입니다.

### [OrderBook](https://github.com/Gabriel-Hong/OrderBook)

C++20으로 구현한 고성능 Limit Order Book 엔진입니다. 주문 처리 레이턴시 200ns를 달성하며, 시스템 프로그래밍과 성능 최적화 역량을 보여줍니다.
